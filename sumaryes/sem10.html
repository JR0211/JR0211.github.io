SEMANA 10

DEFINICIONES

- UseState:
Es un hook que te permite añadir estado a tus componentes funcionales. Antes de la introducción de los hooks en React (versiones anteriores a la 16.8), los componentes funcionales no tenían la capacidad de manejar estados locales.
- JavaScript:
Es un lenguaje de programación de alto nivel, interpretado y orientado a objetos, diseñado principalmente para ejecutarse en navegadores web. Se utiliza para mejorar la interactividad de las páginas web al permitir la manipulación dinámica del contenido, responder a eventos del usuario y realizar operaciones en el lado del cliente. JavaScript es esencial en el desarrollo web moderno y se emplea para crear experiencias interactivas, validar formularios, gestionar el DOM y comunicarse con servidores a través de tecnologías como AJAX.
- Renderizado:
El proceso por el cual React actualiza la interfaz de usuario para reflejar los cambios en el estado de la aplicación, utilizando el método ReactDOM.render().
- API:
Conjunto de reglas y definiciones que permiten que diferentes aplicaciones se comuniquen entre sí. Una API define cómo los componentes de software deben interactuar, qué operaciones pueden realizar y qué datos pueden intercambiar. Las APIs pueden ser utilizadas para permitir la comunicación entre aplicaciones, servicios web, bibliotecas, o cualquier otro tipo de software.
- Axios:
Es una interfaz de procesamiento de aplicaciones entre un servidor web y un navegador web, para el ejemplo se uso una API en especifico.
- Desarrollador Backend:
Es un profesional de la programación que se especializa en la creación y mantenimiento de la lógica, la base de datos y el servidor de una aplicación. A diferencia de los desarrolladores Frontend, que se centran en la interfaz de usuario y la interacción directa con los usuarios, los desarrolladores Backend trabajan en la "parte trasera" o el lado del servidor de una aplicación.

PROCEDIMIENTOS
En el desarrollo de React, se adquirieron conocimientos fundamentales al emplear el hook useState, lo que me permitió incorporar variables de estado en mis componentes para lograr la persistencia del estado entre renderizaciones y la actualización dinámica de la interfaz de usuario. Además, exploré el uso de JSX, una extensión de JavaScript utilizada en React para organizar la interfaz mediante componentes reutilizables. Se profundizó en el entendimiento del proceso de renderizado en React, utilizando ReactDOM.render() para mostrar elementos HTML en la aplicación. También, se adquirió la habilidad de gestionar la transferencia de datos entre componentes, ya sea de manera unidireccional (de padre a hijo o viceversa) o entre componentes anidados.
