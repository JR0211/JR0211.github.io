DEFINICIONES
- Hook:
Se utiliza para referirse a un mecanismo o técnica que permite interceptar eventos o funciones en un sistema. Estos "hooks" son utilizados para modificar o extender el comportamiento de un programa, sistema operativo o aplicación.
- Eventos:
Suceso o acción que ocurre en un sistema y que puede ser detectado, procesado o respondido por un programa o aplicación. Estos eventos pueden ser generados por hardware, software, o por la interacción del usuario con una interfaz gráfica o un dispositivo de entrada.
- Condicionales:
Al uso de estructuras de control de flujo para renderizar o mostrar diferentes elementos o componentes en función de ciertas condiciones. Estas condiciones pueden depender de datos dinámicos, estados de componentes, o cualquier otra lógica que determine qué contenido debe mostrarse en la interfaz de usuario.
- Bucles:
Son estructuras de control de flujo que permiten ejecutar un bloque de código repetidamente mientras se cumple una condición específica. En JavaScript, que es el lenguaje de programación utilizado con React, existen principalmente dos tipos de bucles: for y while.
- Formularios:
Son componentes que permiten a los usuarios ingresar y enviar datos. Los formularios en React operan de manera similar a los formularios en HTML, pero con algunas diferencias notables debido al modelo de datos unidireccional de React y al manejo del estado.
- API:
Conjunto de reglas y definiciones que permiten que diferentes aplicaciones se comuniquen entre sí. Una API define cómo los componentes de software deben interactuar, qué operaciones pueden realizar y qué datos pueden intercambiar. Las APIs pueden ser utilizadas para permitir la comunicación entre aplicaciones, servicios web, bibliotecas, o cualquier otro tipo de software.

.::: USESTATE :::.
Es un Hook de React que te permite agregar una variable de estado a tu componente.
¿Que ofrece UseState?
1. Una variable de estado para mantener los datos entre renderizados.
2. Una función que setea el estado para actualizar la variable y provocar que React renderice el componente nuevamente.
